# Logos des compagnies (PDF des relevés)

Déposer ici un **PNG** par compagnie. Le nom du fichier = le nom de la
compagnie (champ `Compagnie` de la liste SharePoint « Projets »), normalisé :
minuscules, sans accents, espaces/caractères spéciaux → `-`.

Fichiers attendus (compagnies actuelles) :

| Compagnie      | Fichier             |
|----------------|---------------------|
| Ondel          | `ondel.png`         |
| Descimco       | `descimco.png`      |
| Quantech       | `quantech.png`      |
| Talvi          | `talvi.png`         |
| Industro-tech  | `industro-tech.png` |

Recommandations :
- **PNG** (fond transparent idéal — rendu sur bande blanche).
- Résolution confortable (hauteur ≥ ~200 px) ; le logo est calé à 14 mm de
  haut dans le PDF, ratio préservé.

Si le fichier d'une compagnie est absent, le PDF est généré **sans** bande logo
(aucune erreur). Ajouter une nouvelle compagnie = déposer son PNG ici (le nom
doit correspondre au slug ci-dessus).
